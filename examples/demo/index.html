xl3-rs demo

Renders the same three benchmark scenarios as crates/xl3-core/examples/bench.rs, but through the WebAssembly artifact loaded into a Web Worker — the main thread stays interactive throughout. The data and template workbooks are generated in JS using xl3 (TS) + exceljs so this page has no fixture dependency.

Tap a button to drive one scenario; the worker reports the median of three runs back to the UI. xl3 (TS) baseline shown for reference.
